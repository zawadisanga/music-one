# CopperFlorence Music App 🎵

A beautiful music and video streaming app with copper flower aesthetic. Download high-quality audio and video content with ease.

## Features
- 🎵 Search songs and videos
- 📥 Download audio/video in high quality
- 🎨 Beautiful copper flower design
- 📱 Responsive layout
- 🚀 Heroku ready

## Deployment to Heroku

1. Install Heroku CLI
2. Login to Heroku: `heroku login`
3. Create Heroku app: `heroku create your-app-name`
4. Deploy: `git push heroku main`
